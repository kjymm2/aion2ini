3. 적용 방법

  1. 게임을 완전히 종료
  2. 윈도우키 + R → %LOCALAPPDATA%\AION2\Saved\Config\Windows 입력
  3. Engine.ini 를 메모장으로 열기
  4. 기존 내용은 그대로 두고, 맨 아래에 복사한 내용 붙여넣기
  5. 저장 후 게임 실행

되돌리기: 붙여넣은 블록을 삭제하고 저장하면 100% 원상복구됩니다.
